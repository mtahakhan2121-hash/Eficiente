# Eficiente

Pixel-close Next.js marketing site implementation based on the provided Eficiente design exports.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- React 19

## Requirements

- Node.js 18+ recommended
- npm

## Setup

Install dependencies:

```bash
npm install
```

## Run Locally

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## Available Commands

Run the dev server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

Run linting:

```bash
npm run lint
```

## Project Structure

- `src/app` — app router pages and global styles
- `src/components` — reusable UI components
- `src/data` — editable static content
- `public/assets` — SVG assets and design graphics
